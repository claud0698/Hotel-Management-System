## Setup Order

For new installations, follow this order:

1. **Start here**: [QUICK_START.md](QUICK_START.md)
2. **Backend details**: [BACKEND_SETUP.md](BACKEND_SETUP.md)
3. **Database choice**:
   - For Supabase: [SUPABASE_SETUP.md](SUPABASE_SETUP.md)
   - For Singapore region: [SUPABASE_SINGAPORE_SETUP.md](SUPABASE_SINGAPORE_SETUP.md)
4. **Authentication**: [AUTH_README.md](AUTH_README.md)

---

## Prerequisites

Before starting any setup guide, ensure you have:
- Python 3.12 or higher
- Node.js 18+ and npm
- Git installed
- A code editor (VS Code recommended)
- Terminal/command line access
